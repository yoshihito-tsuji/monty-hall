# モンティホール問題 体験シミュレーター

「扉を変えるべきか、変えないべきか」で知られるモンティホール問題を、ブラウザ上で体験できる教材です。

**▶ 実際に遊ぶ： https://yoshihito-tsuji.github.io/monty-hall/**

## できること

- **手で遊ぶ** — 3つの扉から1つを選び、司会者がハズレの扉を開けたあと、「変える」「変えない」を自分で決めて結果を確かめられます。画面上部の4ステップ表示と司会者の吹き出しが、いま何をする場面かを案内します。
- **記録を見る** — 「変えた場合」と「変えなかった場合」のあたった割合を、遊ぶたびに表へ積み上げます。
- **コンピュータにたくさん試してもらう** — 10回・100回・1000回の一括シミュレーションを実行し、2つの作戦のあたる割合が 66.7% と 33.3% に近づいていく様子を棒グラフで確認できます。
- **理由を知る** — 「なぜ変えたほうがあたりやすいのか」を、3つの場合分けの図で説明します。
- **扉の数を増やす** — 扉を3枚・5枚・10枚・100枚（自分で3〜200枚を指定も可）に変えて遊べます。100枚のうち司会者が98枚を開けてみせる様子を見ると、変えたほうがよい理由が直感的につかめます。司会者の開け方も「閉じた扉が残り1枚になるまで開ける」と「ヤギの扉を1枚だけ開ける」から選べ、それぞれの理論値（前者は (N-1)/N、後者は (N-1)/(N(N-2))）と実測を見くらべられます。

## 対応言語

画面右上の「言語」ボタンから、12言語を切り替えられます。

日本語 ／ English ／ 简体中文 ／ 繁體中文 ／ 한국어 ／ Русский ／ Español ／ ไทย ／ हिन्दी ／ Français ／ العربية ／ Svenska

- 選んだ言語はブラウザに記憶され、次に開いたときも同じ言語で表示されます。
- URL に `?lang=en` のように付けると、その言語で直接開けます（例： `https://yoshihito-tsuji.github.io/monty-hall/?lang=fr`）。授業で特定の言語を配りたいときに使えます。
- 初回は、URL の指定 → 前回選んだ言語 → ブラウザの言語設定 → 日本語、の順に決まります。
- アラビア語では、画面全体が右から左に読む配置に切り替わります。

## 使い方

`index.html` をブラウザで開くだけで動きます。インストールや通信は不要です（単一HTMLファイル、外部ライブラリなし）。

## 翻訳を直すには

翻訳の正本は `i18n/<言語コード>.json` です。編集したあと、次のコマンドで `index.html` に反映します。

```bash
python3 tools/build_i18n.py
```

日本語版とキー・プレースホルダ（`{n}` など）・HTMLタグが食い違っていると、エラーを表示して書き込みを中止します。
